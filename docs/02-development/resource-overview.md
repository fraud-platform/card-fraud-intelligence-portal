# Resources Overview

Reference for UI resources and their purpose.

## Rule Management

- `rule-fields` (`/rule-fields`): field metadata and operators
- `rules` (`/rules`): rule identities and versions
- `rulesets` (`/rulesets`): deployment bundles of rule versions
- `approvals` (`/approvals`): maker-checker decisions
- `audit-logs` (`/audit-logs`): compliance trail

## Fraud Operations

- `analyst-home` (`/analyst-home`): analyst landing page
- `worklist` (`/worklist`): queue and claim flows
- `transactions` (`/transactions`): transaction investigation
- `cases` (`/cases`): case lifecycle and activity
- `transaction-metrics` (`/transaction-metrics`): operations dashboard
